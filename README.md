## How to Use

1. Work through lessons in order — each builds on the last.
2. **Do the exercises.** Reading code ≠ writing code.
3. After each session, mark lessons done and push:

```bash
python python-course/tracker.py done 1
git add python-course/progress.json
git commit -m "Completed lesson 1"
git push
```

Progress syncs across machines via GitHub.
